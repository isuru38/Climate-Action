2. Technical Description

For each page implemented (excluding the editor's page and validation page), I utilized modern web development techniques:

HTML Structure: Semantic HTML5 elements including header, nav, main, section, and footer for proper document structure and accessibility.

CSS Styling: Advanced CSS techniques including flexbox, grid layouts, gradient backgrounds, smooth transitions, and responsive design principles.

Key Features: Fixed navigation header, interactive hover effects, card-based layouts, and mobile-responsive design.

Key HTML Tags and CSS Properties

HTML Structure Elements

  • <header> - Site navigation and branding
  • <nav> - Main navigation menu
  • <main> - Primary content area
  • <section> - Content sections (hero, SDG, content grid)
  • <article> - Independent content blocks
  • <footer> - Site footer with links

CSS Properties and Techniques

  • Flexbox: Header layout and navigation alignment
  • CSS Grid: Content card responsive layouts
  • Linear Gradients: Background styling for headers and sections
  • Transitions: Smooth hover effects and animations
  • Box Shadow: Card depth and visual hierarchy
  • Media Queries: Responsive design for mobile devices

Challenges and Lessons Learned

Implementation Challenges

Challenge: Creating a cohesive climate action theme across multiple pages while maintaining usability and accessibility.

Resolution: Developed a consistent color scheme based on environmental themes (blues and greens), used semantic HTML for accessibility, and implemented responsive design principles.


Lesson Learned: The importance of semantic HTML structure not only for accessibility but also for easier CSS styling and maintenance. Using CSS custom properties for consistent theming across pages proved invaluable.

Accessibility

Accessibility Features

All pages demonstrate understanding of accessibility principles:

  • Semantic HTML structure with proper heading hierarchy
  • Alt text for all images describing climate action content
  • Sufficient color contrast ratios for text readability
  • Keyboard navigation support for interactive elements
  • Responsive design ensuring usability across devices
  • Clear focus indicators for navigation elements

All created pages comply with Janet regulations governing web page publication:
