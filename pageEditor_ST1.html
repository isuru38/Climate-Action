Splash Screen Implementation

Technical Description

HTML Meta Redirect: Used <meta http-equiv="refresh" content="4;url=home.html"> to automatically redirect users to the home page after 4 seconds, providing a seamless transition.
CSS Background Scaling: Implemented background-size: cover and background-attachment: fixed to create an immersive full-screen climate action background image.
CSS Animations: Created @keyframes fadeIn and slideInUp animations for smooth element appearance with different animation delays for cascading effects.
Loading Animation: Designed a spinning earth icon using CSS transform: rotate() with infinite animation duration to indicate page loading.

Key HTML Tags Used

  • <meta> - For automatic redirection functionality
  • <header> - Semantic structure for main title area
  • <main> - Primary content container
  • <section> - Grouped related content logically

Key CSS Properties

  • animation: Applied to create smooth transitions and loading effects
  • background: Linear gradients and image positioning for visual appeal
  • transform: Used for rotation and translation effects
  • position: absolute: For precise element positioning

Volunteer Page Implementation

Technical Description

Scrollable Information Section: Implemented using overflow-y: auto and max-height properties to create a contained scrollable area with custom scrollbar styling.
Card-Based Layout: Used CSS Grid (display: grid) with responsive columns (grid-template-columns: repeat(auto-fit, minmax(300px, 1fr))) for adaptive card arrangements.
Interactive Buttons: Applied CSS hover pseudo-class with transition properties for smooth color changes, scaling effects, and box-shadow enhancements.
Dropdown Styling: Styled <select> elements with custom appearance and hover effects, though functionality is decorative as specified.
Form Implementation: Created structured feedback form using <form>, <textarea>, and <input> elements with CSS styling for user interaction.

Key HTML Tags Used

  • <section> - Organized content into logical sections
  • <article> - Individual volunteer program cards
  • <form> - User feedback submission structure
  • <textarea> - Multi-line text input for reviews
  • <select> - Dropdown for filtering options

Content Page Implementation

Technical Description

Internal Navigation: Implemented anchor links (<a href="#section-id">) connecting to elements with corresponding id attributes for smooth scrolling navigation.
Fixed "Go to Top" Button: Used position: fixed with bottom and right positioning, ensuring constant visibility. CSS hover effects enhance user interaction.
Hierarchical Headings: Properly structured content using <h1> through <h6> tags following semantic HTML principles for accessibility and SEO.
Responsive Images: Applied max-width: 100% and height: auto for responsive image scaling, with CSS border-radius and box-shadow for visual enhancement.

Challenges and Solutions

Challenge: Automatic Redirect Timing

Issue: Balancing user experience with sufficient time to read splash screen content.

Solution: Implemented 4-second delay with visual loading indicator to give users adequate time while maintaining engagement through animated elements.

Challenge: Responsive Card Layout

Issue: Creating volunteer program cards that adapt to different screen sizes effectively.

Solution: Used CSS Grid with minmax() function and auto-fit to create flexible, responsive layouts that maintain visual hierarchy across devices.

Challenge: Cross-browser Compatibility

Issue: Ensuring animations and transitions work consistently across different browsers.

Solution: Used vendor prefixes (-webkit-, -moz-) and tested fallback styles for older browser support.

Lessons Learned

  • Importance of progressive enhancement in web design
  • Value of semantic HTML for accessibility and SEO
  • CSS Grid provides superior layout control over floats
  • Animation timing is crucial for user experience
  • Testing across multiple devices is essential
