Accessibility

Implemented accessibility features across all pages:

  • Semantic HTML5 structure (header, nav, main, footer)
  • Alt text for all images
  • Keyboard-navigable interface
  • Color contrast ratio > 4.5:1 for readability
  • ARIA labels where needed

Challenges & Lessons Learned

Key challenges faced during development:

  • Responsive Tables: Making the data table readable on mobile required careful media query implementation
  • Image Paths: Ensuring image paths worked correctly across different devices
  • CSS Organization: Maintaining clean and reusable CSS across multiple pages

Key Lessons:

  • Importance of mobile-first development approach
  • Benefits of semantic HTML for accessibility
  • Value of consistent file organization
  • Effective use of CSS Grid and Flexbox
